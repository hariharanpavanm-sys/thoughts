Workflow Guide

Follow these steps to update your private blog:

  1. Open the local file _chat.txt in your workspace.
  2. Write your new thoughts, notes, or append chat messages.
  3. Save the file and run the encryption command locally: 
    python encrypt.py
  4. Commit the updated posts.json.enc file to your public GitHub repository and push: 
    git add posts.json.enc
git commit -m "Update thoughts"
git push
  5. Your live site will fetch the new thoughts and decrypt them dynamically using your password!
